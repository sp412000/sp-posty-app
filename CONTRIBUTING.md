# Contributing

Thanks for helping improve this project.

## Adding a new post

1. Create a new Markdown file in the `posts/` folder.
2. Use a clear title in the first line, for example:
   ```md
   # My New Post
   ```
3. Write the post content below the title.
4. Save the file with a descriptive filename and the `.md` extension.
5. Start the app locally and verify the new post appears in the list.

## General guidelines

- Keep content clear and concise.
- Use descriptive filenames and friendly titles.
- Test your changes locally before submitting a pull request.
